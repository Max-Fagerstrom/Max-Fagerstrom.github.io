ShutterSeeker

This project was a game project for my education at Futuregames, i worked on it for 6 weeks in a group of 13 people. My work on this project included programming of gameplay systems like a Day/Night cycle, and backend systems which in this game included analysis and scoring of player taken pictures, we did this using the OpenCV library.

Itch.io

Photo Analysis

The most fun part to work on during this project was the photo analysis system, it uses OpenCV to compare player taken pictures to a library of pictures with associated properties that is taken into account in the code that gives the photo a grade.

The system also checks for photo blurriness and similarity to already taken pictures, and deducts points in this case.
